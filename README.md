# Olist-E-Commerce-Data-Analytics-Project
 10K+ yorumda NLP ile duygu analizi yaparak lojistik kaynaklı memnuniyetsizliği saptadım. A/B testleriyle 10 günü aşan gecikmelerin puanları 4.29'dan 3.83'e düşürdüğünü kanıtladım.  RFM &amp; Tahminleme: Müşterileri 10 segmente ayıran RFM modeli geliştirdim. Zaman serisi analiziyle Black Friday yoğunluğunu saptayıp stok yönetimi önerileri sundum.
